# M5OS
A firmware for the M5StickS3 with app support (in .py) with builtin system APIs and UI kits.
# Development (packaging the firmware)
There are 2 .py files: pack.py and png_to_bmp.py. These will help you create firmware updates, UI icons and app icons.
